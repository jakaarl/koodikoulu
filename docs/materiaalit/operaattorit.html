## Operaattorit [Operaattori](https://en.wikipedia.org/wiki/Operator_%28computer_programming%29) muokkaa muuttujaa. Muuttujien yhteydessä käytimme jo sijoitusoperaattoria ([assignment](https://en.wikipedia.org/wiki/Assignment_%28computer_science%29) operator): ``` var foo = "bar"; ``` Sijoitusoperaattori on Javascriptissä yhtäsuuruusmerkki, `=`. Se siis ei vertaa, vaan sijoittaa muuttujalle arvon! Mistä pääsemmekin seuraavaan operaattoriin, vertailuun: `==`: ``` var samansuuruinen = (foo == bar); ``` Yllä muuttuja "samansuuruinen" on `true`, mikäli muuttujat "foo" ja "bar" ovat samansuuruisia, ja `false` jos ne ovat erisuuruisia. --- [<< sisällysluettelo](index.html)
