# Javascript [JavaScript](https://fi.wikipedia.org/wiki/JavaScript) julkaistiin alkujaan vuonna 1995 [Netscape Navigator](https://fi.wikipedia.org/wiki/Netscape_Navigator) -selaimen mukana. Se mahdollisti staattisten [HTML-webbisivujen](https://fi.wikipedia.org/wiki/World_Wide_Web) elävöittämisen interaktiivisella toiminnallisuudella. Nykyisin JS on standardoitu [ECMAScript](https://www.ecma-international.org/ecma-262/) -standardilla. JS on myös löytänyt tiensä ulos selaimesta: [Node.js](https://nodejs.org/) mahdollistaa sen käyttämisen palvelinohjelmoinnissa ja Java-kielen ohjelmointirjastoon sisältyy [Nashorn -JavaScript-tulkki](https://en.wikipedia.org/wiki/Nashorn_(JavaScript_engine)). JavaScript on yksi yleisimmin käytetyistä ohjelmointikielistä ja sen käyttäminen on yksinkertaisempaa, kuin monien muiden ohjelmointikielten. JS-tulkki sisältyy kaikkiin moderneihin webbiselaimiin, joten erillistä asennusta ei tarvita. Useimmissa selaimissa on mukana kehitystyökaluja - esimerkiksi [Firefoxissa](https://www.mozilla.org/fi/firefox/) ja [Chromessa](https://www.google.fi/intl/fi/chrome/browser/desktop/) näppäinyhdistelmä `Ctrl-Shift-I` avaa kehitystyökalut. ## JS-lähdekoodi Tavallisesti JS-kooditiedoston tiedostopääte on `.js`, mutta koodi vois myös olla suoraan webbisivun HTML-tiedostossa. Yleensä on kuitenkin parempi pitää etenkin pidemmät ohjelmat omissa tiedostoissaan HTML-tiedostojen sijaan. Periaatteessa lähdekoodia voi kirjoittaa ja muokata millä tahansa tekstieditorilla, mutta erityisesti koodaamiseen tarkoitetuissa editoreissa on paljon hyödyllisiä ominaisuuksia. Joitakin ilmaisia editoreita, jotka soveltuvat JS-koodaamiseen: * [Atom](https://atom.io) * [Brackets](http://brackets.io/) * [Light Table](http://lighttable.com/) * [Programmer's Notepad](http://www.pnotepad.org/) * [Visual Studio Code](https://code.visualstudio.com/) On olemassa myös selaimessa toimivia editoreita, jotka eivät vaadi asennusta, esim: * https://jsfiddle.net/ * https://jsbin.com/ * https://js.do/ * https://codeanywhere.com/ * https://repl.it/languages/javascript ## Hei maailma! Ohjelmoinnissa on ikiaikainen perinne aloittaa ["Hei maailma!"](https://fi.wikipedia.org/wiki/Hei_maailma_-ohjelma) -ohjelmalla (_"Hello world!"_).
