## Ehtolauseet Ehtolauseessa arvioidaan (_evaluate_) lausekkeen totuusarvo, ja sen ollessa tosi, suoritetaan seuraava lause (_statement_) tai lohko (_block_). Tavallisin ehtolause on `if`: ``` if (muuttuja > 0) { console.log("muuttuja on positiivinen."); } ``` Mikäli "muuttuja" on arvoltaan suurempi, kuin 0, lokitetaan konsoliin. Aaltosulkeet `{` ja `}` aloittavat ja lopettavat lohkon. Kun ehtolauseen jälkeen halutaan suorittaa enemmän, kuin yksi lause, aaltosulkeet ovat pakollisia - muuten ehdollisuus koskisi vain ensimmäistä lausetta. Virheiden välttämiseksi onkin parempi käyttää aina aaltosulkeita! ``` if (muuttuja > 0) console.log("muuttuja on..."); // suoritetaan, jos muuttuja on suurempi kuin nolla... console.log("positiivinen!"); // hups, tämä suoritetaankin aina! ``` If-lausetta käyttäessä tarvitsee usein hanskata myös vaihtoehto, että lause on epätosi. Tähän soveltuu `else`: ``` if (muuttuja > 0) { console.log("muuttuja on positiivinen."); } else { console.log("muuttuja on negatiivinen."); } ``` Yllä suoritetaan `else`, mikäli "muuttuja" ei ole nollaa suurempi.
